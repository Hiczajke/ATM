║                          PRZEPŁYW DANYCH                                       ║

1. LOGOWANIE
   Użytkownik → GUI (PySide6) → Autentykacja → SQLite (clients)
                                    ↓
                            Log (atmlog-admin.log)

2. WYPŁATA
   Użytkownik → GUI → ATM.withdraw() → Algorytm Backtracking
                          ↓
                      SQLite (banknote_balance, transactions)
                          ↓
                      Log (atmlog-withdraws.log + atmlog-transactions.log)

3. WPŁATA
   Użytkownik → GUI → ATM.deposit() → Aktualizacja stanów
                          ↓
                      SQLite (banknote_balance, overflow_balance, transactions)
                          ↓
                      Log (atmlog-deposits.log + atmlog-transactions.log)

4. OPERACJE ADMIN
   Admin → GUI → Zarządzanie banknotami → SQLite
                          ↓
                      Log (atmlog-admin.log)
